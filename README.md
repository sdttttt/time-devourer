
☢️该项目是基于`Win32API(非MFC)`编写(学习目的),不可能避免会导致以下情况：
 - 内存泄漏
 - 致命错误
 - UI错误
 - 进程装死


### Tip

由于只用了Win32API，该程序资源占用极小。

这个项目只用了`Win32API`和GDI, 理论上可以在`Windows 2000`上运行。
